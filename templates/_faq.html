Does PasswordProtectMyFile.com ever see my file?

Absolutely not. All the encryption and decryption happens via client-side javascript on your computer.

Is the code open source?

Check it out on github

How is my file encrypted?

The encryption is AES using an OpenSSL-compatible salt and key derivation function. The core crypto implementation is from the open-source google library CryptoJS.

Can I encrypt or decrypt from the command line?

Yes! There's an npm package implementing encryption and decryption. npm install -g ppmf && ppmf --help

What browsers are supported?

All modern browsers are supported. This includes Chrome, Firefox, Opera, Internet Explorer 10+, and Safari 6.1+.
